Exercício IW2 - Aula 20/03/2025

Criar um programa que receba a nota de um aluno e exiba a mensagem correspondente:

Nota >= 7 → "Aprovado ✅"
Nota entre 5 e 6.9 → "Recuperação ⚠️"
Nota < 5 → "Reprovado ❌"

Desafio extra:

Modifique o código para permitir que o usuário digite a nota. Para isso, use prompt().
